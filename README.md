# go-lab
Uma coleção de experimentos, exercícios e projetos práticos em Go para dominar a linguagem.

1. Objetivo do Repositório
Praticar a sintaxe de Go.
Explorar bibliotecas padrão e externas.
Documentar a evolução no aprendizado de backend.

2. Roadmap de Estudos (Checklist)
[x] Variáveis e Tipos de Dados
[x] Estruturas de Controle (if, for, switch)
[x] Structs e Interfaces
[ ] Concorrência (Goroutines e Channels)
[ ] Manipulação de JSON e APIs HTTP

3. Organização do Projeto
/basics: Exercícios iniciais de sintaxe.
/challenges: Resolução de problemas de lógica.
/projects: Mini-aplicações (Uma CLI ou um servidor web simples).

4. Como Executar
Para rodar um arquivo específico
go run main.go

Para rodar os testes
go test ./...
